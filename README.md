# NeoMate AI

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.13+](https://img.shields.io/badge/python-3.13+-blue.svg)](https://www.python.org/downloads/)

**আপনার ব্যক্তিগত, স্বশাসিত ডেস্কটপ AI সহকারী, যা দেখতে, শুনতে এবং কাজ করতে পারে।**

NeoMate AI একটি অত্যাধুনিক, পূর্ণাঙ্গ ডেস্কটপ অটোমেশন এজেন্ট যা মানুষের মতো কম্পিউটার পরিচালনা করতে সক্ষম। এটি ভয়েস কমান্ড, স্ক্রিন ইন্টারঅ্যাকশন, এবং LLM-ভিত্তিক ডিসিশন মেকিং এর মাধ্যমে যেকোনো জটিল কাজ দক্ষতার সাথে সম্পাদন করে।

## প্রধান বৈশিষ্ট্য

- 🧠 **মানবসদৃশ বুদ্ধিমত্তা:** জটিল কাজ বোঝা, পরিকল্পনা করা এবং স্ব-সংশোধন করার ক্ষমতা।
- 👁️ **ভিজ্যুয়াল পারসেপশন:** স্ক্রিন 'দেখে' যেকোনো সফটওয়্যার বা ওয়েবসাইট নিয়ন্ত্রণ করার ক্ষমতা।
- 🗣️ **ভয়েস ইন্টিগ্রেশন:** প্রাকৃতিক ভাষায় কথা বলা ও শোনার মাধ্যমে সম্পূর্ণ হ্যান্ডস-ফ্রি অপারেশন।
- 🔒 **প্রাইভেসি-ফার্স্ট ডিজাইন:** প্রায়-সম্পূর্ণ অফলাইন আর্কিটেকচার যা আপনার ডেটা নিরাপদ রাখে।
- 🚀 **মাল্টি-টাস্কিং:** একাধিক কাজ একসাথে (যেমন, ফাইল ম্যানেজমেন্ট ও গান শোনা) করার ক্ষমতা।
- 🔄 **সেলফ-লার্নিং ও সেলফ-করেকশন:** নিজের কাজের উন্নতি সাধন ও ভুল সংশোধনের ক্ষমতা।
- 🌐 **বহুভাষিক সাপোর্ট:** বাংলা, ইংরেজি সহ বিভিন্ন ভাষায় কাজ করার সক্ষমতা।

## আর্কিটেকচার

NeoMate AI একটি মডুলার আর্কিটেকচারে তৈরি, যা নিম্নলিখিত কম্পোনেন্টগুলো নিয়ে গঠিত:

- **ভয়েস ইনপুট মডিউল:** Whisper ব্যবহার করে ভয়েস কমান্ড গ্রহণ করে।
- **ভিজ্যুয়াল প্রসেসিং মডিউল:** OpenCV এবং PyAutoGUI ব্যবহার করে স্ক্রিন ইন্টারঅ্যাকশন।
- **LLM ইঞ্জিন:** Ollama এর মাধ্যমে লোকাল LLM ব্যবহার করে ডিসিশন মেকিং।
- **টাস্ক ম্যানেজমেন্ট:** AutoGen ব্যবহার করে মাল্টি-টাস্কিং।
- **মেমোরি স্টোরেজ:** SQLite ব্যবহার করে লোকাল ডেটা স্টোরেজ।
- **ভয়েস আউটপুট মডিউল:** Coqui TTS ব্যবহার করে ভয়েস রেসপন্স।

## শুরু করার গাইড

### পূর্বশর্ত

- Python 3.10 বা তার উপরে
- Git
- Ollama (লোকাল LLM রান করার জন্য)

### ইনস্টলেশন

```bash
git https://github.com/emonhmamun/NeoMate-Agent-AI.git
cd NeoMate-AI
python -m venv NeoMate_env
# Windows
NeoMate_env\Scripts\activate
# Linux/Mac
source NeoMate_env/bin/activate
pip install -r requirements.txt
```

### কনফিগারেশন

`config/settings.yaml` ফাইলটি খুলে আপনার পছন্দ অনুযায়ী সেটিংস পরিবর্তন করুন। যেমন ভাষা, মডেল পাথ, সিকিউরিটি অপশন ইত্যাদি।

### চালানো

```bash
python src/main.py
```
