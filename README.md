# NeoMate AI

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.13+](https://img.shields.io/badge/python-3.13+-blue.svg)](https://www.python.org/downloads/)

**আপনার ব্যক্তিগত, স্বশাসিত ডেস্কটপ AI সহকারী, যা দেখতে, শুনতে এবং কাজ করতে পারে।**

NeoMate AI একটি অত্যাধুনিক, পূর্ণাঙ্গ ডেস্কটপ অটোমেশন এজেন্ট যা মানুষের মতো কম্পিউটার পরিচালনা করতে সক্ষম। এটি ভয়েস কমান্ড, স্ক্রিন ইন্টারঅ্যাকশন, এবং LLM-ভিত্তিক ডিসিশন মেকিং এর মাধ্যমে যেকোনো জটিল কাজ দক্ষতার সাথে সম্পাদন করে। ব্যবহারকারী ইচ্ছা অনুযায়ী এটি অনলাইন বা অফলাইন মোডে কাজ করতে পারে, যা সর্বোচ্চ নমনীয়তা এবং গোপনীয়তা নিশ্চিত করে।

## প্রধান বৈশিষ্ট্য

- 🧠 **মানবসদৃশ বুদ্ধিমত্তা:** জটিল কাজ বোঝা, পরিকল্পনা করা এবং স্ব-সংশোধন করার ক্ষমতা, যা AI কে স্বয়ংসম্পূর্ণ করে তোলে।
- 👁️ **ভিজ্যুয়াল পারসেপশন:** স্ক্রিন 'দেখে' যেকোনো সফটওয়্যার বা ওয়েবসাইট নিয়ন্ত্রণ করার ক্ষমতা, যা ব্যবহারকারীর ইন্টারফেসের সাথে স্বয়ংক্রিয় ইন্টারঅ্যাকশন নিশ্চিত করে।
- 🗣️ **ভয়েস ইন্টিগ্রেশন:** প্রাকৃতিক ভাষায় কথা বলা ও শোনার মাধ্যমে সম্পূর্ণ হ্যান্ডস-ফ্রি অপারেশন, যা ব্যবহারকারীর অভিজ্ঞতা উন্নত করে।
- 🔒 **প্রাইভেসি-ফার্স্ট ডিজাইন:** প্রায়-সম্পূর্ণ অফলাইন আর্কিটেকচার যা আপনার ডেটা নিরাপদ রাখে এবং গোপনীয়তা বজায় রাখে।
- 🌐 **অনলাইন ও অফলাইন মোড:** ব্যবহারকারীর চাহিদা অনুযায়ী অনলাইন বা অফলাইন মোডে কাজ করার নমনীয়তা।
- 🚀 **মাল্টি-টাস্কিং:** একাধিক কাজ একসাথে (যেমন, ফাইল ম্যানেজমেন্ট, গান শোনা, ওয়েব ব্রাউজিং) করার ক্ষমতা।
- 🔄 **সেলফ-লার্নিং ও সেলফ-করেকশন:** AI নিজেই নিজের কাজের উন্নতি সাধন ও ভুল সংশোধনের ক্ষমতা রাখে।
- 🌍 **বহুভাষিক সাপোর্ট:** বাংলা, ইংরেজি সহ বিভিন্ন ভাষায় কাজ করার সক্ষমতা, যা বৈশ্বিক ব্যবহার নিশ্চিত করে।
- ⚙️ **কাস্টমাইজেশন:** ব্যবহারকারীর চাহিদা অনুযায়ী সহজে কনফিগার এবং এক্সটেন্ড করা যায়।

## প্রযুক্তি স্ট্যাক

| প্রযুক্তি | বর্ণনা |
|----------|--------|
| Python | মূল প্রোগ্রামিং ভাষা, দ্রুত এবং বহুমুখী |
| PyAutoGUI | ডেস্কটপ অটোমেশন এবং GUI ইন্টারঅ্যাকশন |
| OpenCV | উন্নত ভিজ্যুয়াল প্রসেসিং এবং ইমেজ রিকগনিশন |
| Ollama | লোকাল LLM রান করার জন্য, গোপনীয়তা রক্ষা করে |
| Whisper | উন্নত ভয়েস রেকগনিশন প্রযুক্তি |
| Coqui TTS | প্রাকৃতিক এবং স্পষ্ট ভয়েস আউটপুট |
| AutoGen | টাস্ক প্ল্যানিং ও ম্যানেজমেন্টের জন্য আধুনিক ফ্রেমওয়ার্ক |
| SQLite | দ্রুত এবং নিরাপদ লোকাল মেমোরি স্টোরেজ |

## আর্কিটেকচার

NeoMate AI একটি মডুলার এবং স্কেলেবল আর্কিটেকচারে তৈরি, যা নিম্নলিখিত প্রধান কম্পোনেন্ট নিয়ে গঠিত:

- **ভয়েস ইনপুট মডিউল:** Whisper ব্যবহার করে উন্নত ভয়েস কমান্ড গ্রহণ এবং প্রক্রিয়াকরণ।
- **ভিজ্যুয়াল প্রসেসিং মডিউল:** OpenCV ও PyAutoGUI এর মাধ্যমে স্ক্রিনের যেকোনো এলিমেন্ট চিনে নিয়ে ইন্টারঅ্যাকশন।
- **LLM ইঞ্জিন:** Ollama এর মাধ্যমে লোকাল LLM ব্যবহার করে দ্রুত এবং নিরাপদ ডিসিশন মেকিং।
- **টাস্ক ম্যানেজমেন্ট:** AutoGen ফ্রেমওয়ার্ক ব্যবহার করে মাল্টি-টাস্কিং এবং টাস্ক অর্গানাইজেশন।
- **মেমোরি স্টোরেজ:** SQLite ডাটাবেস ব্যবহার করে লোকাল ডেটা স্টোরেজ এবং রিট্রিভাল।
- **ভয়েস আউটপুট মডিউল:** Coqui TTS ব্যবহার করে প্রাকৃতিক এবং স্পষ্ট ভয়েস রেসপন্স জেনারেশন।

## ডেমো

NeoMate AI এর কিছু কার্যকরী ডেমো উদাহরণ:

### ভয়েস কমান্ড উদাহরণ

- "ফাইল খোলো" - নির্দিষ্ট ফাইল বা ফোল্ডার খুলতে।
- "গান চালাও" - পছন্দের মিউজিক প্লেয়ার চালু করতে।
- "স্ক্রিনশট নাও" - সম্পূর্ণ স্ক্রিন বা নির্দিষ্ট অংশের স্ক্রিনশট নিতে।
- "ব্রাউজার খোলো এবং গুগল সার্চ করো" - ওয়েব ব্রাউজার চালু করে গুগলে সার্চ করতে।

### স্ক্রিন ইন্টারঅ্যাকশন উদাহরণ

NeoMate AI স্ক্রিনের যেকোনো এলিমেন্ট চিনে নিয়ে স্বয়ংক্রিয়ভাবে ক্লিক, ড্র্যাগ, টেক্সট ইনপুট ইত্যাদি করতে সক্ষম।

### মাল্টি-টাস্কিং ডেমো

একসাথে ফাইল ম্যানেজমেন্ট, মিউজিক প্লেব্যাক এবং ওয়েব ব্রাউজিং পরিচালনা করার ক্ষমতা প্রদর্শন।

## শুরু করার গাইড

### পূর্বশর্ত

- Python 3.10 বা তার উপরে ইনস্টল থাকতে হবে।
- Git ইনস্টল থাকতে হবে।
- Ollama লোকাল LLM রান করার জন্য ইনস্টল ও কনফিগার থাকতে হবে।

### ইনস্টলেশন

```bash
git clone https://github.com/emonhmamun/NeoMate-Agent-AI.git
cd NeoMate-AI
python -m venv NeoMate_env
# Windows
NeoMate_env\Scripts\activate
# Linux/Mac
source NeoMate_env/bin/activate
pip install -r requirements.txt
```

### কনফিগারেশন

`config/settings.yaml` ফাইলটি খুলে ভাষা, মডেল পাথ, সিকিউরিটি অপশন ইত্যাদি আপনার প্রয়োজন অনুযায়ী কাস্টমাইজ করুন।

### চালানো

```bash
python src/main.py
```

- **Ollama না চললে:** Ollama সঠিকভাবে ইনস্টল ও রান হচ্ছে কিনা নিশ্চিত করুন।

- **পাইথন ভার্সন সমস্যা:** Python 3.10 বা তার উপরে ভার্সন ব্যবহার করুন।
- **ডিপেন্ডেন্সি ইনস্টলেশন সমস্যা:** ভার্চুয়াল এনভায়রনমেন্ট সক্রিয় করে ইনস্টল করুন।

### লগ চেক করুন

ত্রুটি বা সমস্যা হলে লগ ফাইল বা কনসোল আউটপুট পরীক্ষা করুন।

## FAQ

**Q: NeoMate AI কি ফ্রি?**  
A: হ্যাঁ, NeoMate AI সম্পূর্ণ ওপেন সোর্স এবং MIT লাইসেন্সের অধীনে মুক্ত।

**Q: কোন ভাষা সাপোর্ট করে?**  
A: বাংলা, ইংরেজি সহ বহুভাষিক সাপোর্ট রয়েছে।

**Q: ইন্টারনেট কানেকশন দরকার?**  
A: NeoMate AI অনলাইন ও অফলাইন দুই মোডেই কাজ করতে পারে, তবে প্রাথমিক সেটআপে ইন্টারনেট দরকার হতে পারে।

**Q: কিভাবে কাস্টমাইজ করব?**  
A: `config/settings.yaml` ফাইল এডিট করে সহজেই কাস্টমাইজেশন করা যায়।
